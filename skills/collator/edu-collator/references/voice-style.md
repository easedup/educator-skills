# Voice & Style Capture

How to learn and apply a teacher's written voice for report comments and
parent-facing communications.

---

## When this is needed

Only invoked for:
- Report Comment Writer
- Parent Comms Writer (optional but preferred)

For all other skills (lesson plans, units, assessments), write in clear
professional educator language — no voice capture needed.

---

## Detecting a loaded voice profile

If a `voice-style` skill is available in the current tool, use it silently.
No need to ask the teacher for examples.

If no voice profile is loaded and the teacher wants report comments or parent
communications, ask once:

> "To match your writing style, it helps to see a few examples of your own
> comments or communications — even rough ones. Could you paste 3–5 examples?
> If not, I'll write in clear, warm professional language."

If they decline, proceed with the default voice (see below).

---

## Extracting a voice profile from examples

When the teacher provides writing samples, analyse for:

**Tone markers**
- Formal ↔ conversational
- Warm ↔ neutral ↔ direct
- Encouraging ↔ matter-of-fact

**Sentence patterns**
- Average sentence length (short and punchy vs. longer and flowing)
- Active vs. passive voice tendency
- Use of the student's name (how often, where in the comment)

**Vocabulary choices**
- Subject-specific terms they favour
- Phrases they repeat (e.g. "demonstrates a solid understanding", "is developing")
- Words or phrases to avoid (note if they never use certain terms)

**Structure patterns**
- How they open a comment (strength first? area of focus first?)
- How they close (forward-looking? encouragement? next steps?)
- Do they use contractions?

Store this as a voice profile for the session.

---

## Applying the voice profile

When generating report comments or parent communications:
- Mirror sentence length and rhythm
- Use the teacher's preferred vocabulary where it fits
- Follow their structural pattern (opening → body → close)
- Apply their warmth level — don't add warmth they didn't show, don't strip
  warmth they did

After generating, note briefly: "I've matched the tone from your examples —
let me know if you'd like it warmer/more direct/shorter."

---

## Default voice (no profile loaded)

Clear, warm, professional Australian educator English (or appropriate regional
variant based on CURRICULUM setting):
- Active voice
- Student named once at the start
- Strength → growth area → forward-looking structure
- No jargon
- Approx. 50–80 words for a standard report comment
