# Communication

Skills for writing teacher-to-family and school communications.

- **[parent-comms](./parent-comms/SKILL.md)** — Drafts polished parent-facing communications (newsletters, curriculum updates, excursion notes, learning updates, end-of-term wrap-ups) in the teacher's own voice.
- **[parent-curriculum-explainer](./parent-curriculum-explainer/SKILL.md)** — Translates curriculum content into plain English for families — what students are learning, why it matters at this stage, and how families can support at home.
- **[report-comment](./report-comment/SKILL.md)** — Drafts personalised report comments in the teacher's own voice, aligned to achievement standards language. The teacher provides the grade judgment — the skill never infers one.
- **[voice-style-capture](./voice-style-capture/SKILL.md)** — Learns a teacher's written voice from examples so that report comments and parent communications sound like the teacher wrote them. Run once to produce a saved voice profile used by Report Comment and Parent Comms.
